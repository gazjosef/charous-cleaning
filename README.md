<h3 align="center">Charous Cleaning</h3>

  <p align="center">
A professional and user-friendly website for a pool cleaning, repairing, and servicing business. This website uses technologies such as Vue, Nuxt, and Tailwind.
    
  <p align="center">Vue is a progressive JavaScript framework that provides a highly reactive and customizable user interface. Nuxt is a higher-level framework that adds server-side rendering, static site generation, and improved performance to Vue applications. Tailwind is a utility-first CSS framework that makes it easy to create fast-loading, highly-readable, and responsive web pages.</p>
